# 아두이노 보드 연동 테스트 명세서

실제 아두이노 보드와 GUI를 연결해 수행하는 테스트 명세입니다.  
프로젝트 흐름은 `project-flow.md`를 참고하세요.

---

## 1. 테스트 환경

| 항목 | 내용 |
|------|------|
| 앱 실행 | `npm run dev` 또는 패키징된 Electron 실행 파일 |
| 통신 | 시리얼 9600 baud, 줄 단위(\r\n) |
| 보드 요구 | 위 프로토콜(GUI 송수신 형식)을 처리하는 펌웨어 업로드 완료 |

---

## 2. 보드 펌웨어 체크리스트 (테스트 전 확인)

GUI가 기대하는 **수신 형식**을 보드가 한 줄씩 출력하는지 확인하세요.

- [ ] 실시간 위치: `CAM_HEIGHT=숫자`, `CAM_LOWER=숫자`, `TABLE_HEIGHT=숫자`
- [ ] 단일 축 완료: `DONE CAM_HEIGHT=숫자` (CAM_LOWER, TABLE_HEIGHT 동일 형식)
- [ ] 자동운전 완료: `DONE_ALL` (뒤에 추가 텍스트 가능)
- [ ] 긴급 정지 완료: `DONE_STOP`
- [ ] 종료 신호: `QUIT` (QUIT_HOME 수신 후 홈 복귀 완료 시 1회)

GUI가 **전송하는 명령**을 보드가 처리하는지 확인하세요.

- [ ] `INIT_CAM_HEIGHT` / `INIT_CAM_LOWER` / `INIT_TABLE_HEIGHT`
- [ ] `FULL_INIT`
- [ ] `CAM_HEIGHT=값`, `CAM_LOWER=값`, `TABLE_HEIGHT=값`
- [ ] `AUTO=CAM_HEIGHT:v1/CAM_LOWER:v2/TABLE_HEIGHT:v3`
- [ ] `STOP`
- [ ] `QUIT_HOME` (처리 후 `QUIT` 전송)
- [ ] `RESET`

## 3. 테스트 케이스

### 3.1 연결/해제

| ID | 시나리오 | 조치 | 예상 결과 |
|----|----------|------|-----------|
| T1.1 | 포트 목록 | 앱 실행 → 포트 선택 드롭다운 또는 갱신 | 사용 가능한 시리얼 포트 목록 표시, 로그에 "포트 목록 갱신됨 (N개)" |
| T1.2 | 연결 | 포트 선택 후 연결 버튼 클릭 | 연결 상태 표시, 선택 포트명 표시, 로그 "포트 XXX 연결됨" |
| T1.3 | 연결 실패 | 존재하지 않거나 사용 중인 포트로 연결 시도 | 로그에 "연결 실패: ..." |
| T1.4 | 해제 | 연결된 상태에서 연결 해제 버튼 클릭 | 연결 해제, 로그 "연결 해제됨", System Status는 Ready |

### 3.2 단축 초기화

| ID | 시나리오 | 조치 | 예상 결과 |
|----|----------|------|-----------|
| T2.1 | 단축 초기화 (연결됨) | 한 축(예: 카메라 높이)에서 "초기화" 클릭 | 로그 "송신: INIT_CAM_HEIGHT", 해당 패널에 로딩/이동 중 표시 |
| T2.2 | 단축 완료 반영 | 보드가 `DONE CAM_HEIGHT=0` 등 전송 | 해당 축 currentValue 갱신, 축 상태 완료, 모든 축 완료 시 System Ready |
| T2.3 | 미연결 시 초기화 | 연결 해제 후 초기화 버튼 클릭(다른 경로로 호출 시) | 로그 "연결되지 않음 - 명령 무시됨", 축이 moving으로 고정되지 않음 |

### 3.6 긴급 정지

| ID | 시나리오 | 조치 | 예상 결과 |
|----|----------|------|-----------|
| T6.1 | 정지 버튼 (Running 중) | 자동운전 또는 축 이동 중 "정지" 클릭 | 로그 "송신: STOP", "긴급 정지!", System Status → Emergency Stop |
| T6.2 | DONE_STOP 수신 | 보드가 `DONE_STOP` 전송 | 로그 "긴급 정지 완료", 모든 축 idle, System Ready |
| T6.3 | 정지 상태 유지 | 긴급 정지 후 보드가 지연된 DONE 한 줄 전송 | System Status가 Emergency Stop으로 유지(ready로 덮어쓰지 않음) |
| T6.4 | 정지 버튼 비활성 | System Ready일 때 | "정지" 버튼 disabled |
